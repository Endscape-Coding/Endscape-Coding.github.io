Наши особенности

Удобство для пользователя

Никаких установок драйверов, монтирования устройств и лишних установок зависимостей: из коробки вы получаете все необходимое для общения, разработки и игр

Интуитивный установщик Calamares делает процесс установки доступным даже для новичков.

Безопасность

Встроенные механизмы безопасности и регулярные обновления для защиты вашей системы.

Высокая производительность

Оптимизированное ядро и минимальные накладные расходы обеспечивают максимальную отзывчивость.

Богатый набор ПО

Создание Live USB

1

Скачайте образ

Загрузите последнюю версию EN-OS 0.8 BETA с нашей страницы релизов.

Скачать EN-OS
2

Запишите на USB

Используйте один из предложенных методов для записи Live-образа на флешку:

Rufus
dd
Ventoy

Rufus (Windows)

  1. Запустите Rufus и выберите вашу USB-флешку
  2. Нажмите 'ВЫБРАТЬ' и укажите скачанный ISO-образ
  3. Оставьте остальные настройки по умолчанию
  4. Нажмите 'СТАРТ' и дождитесь завершения
Скачать Rufus

dd (Linux/macOS)

  1. Определите путь к вашему USB-устройству (например, /dev/sdX)
  2. Размонтируйте устройство, если оно смонтировано
  3. Выполните команду записи:
dd if=en-os-0.6-beta.iso of=/dev/sdX bs=4M status=progress oflag=sync

ВАЖНО: Замените /dev/sdX на фактический путь к вашему устройству. Неправильное использование может привести к потере данных!

Ventoy (Multi-Platform)

  1. Скачайте и установите Ventoy на вашу USB-флешку
  2. Скопируйте ISO-образ EN-OS на флешку
  3. Загрузитесь с USB и выберите образ EN-OS в меню Ventoy
