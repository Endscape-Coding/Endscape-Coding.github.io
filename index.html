EN-OS

Современный дистрибутив на базе Arch Linux с предустановленным KDE Plasma, оптимизированный для производительности и удобства прямо из коробки без каких либо дополнительных настроек.

Наши особенности

Удобство для пользователя

Никаких установок драйверов, монтирования устройств и лишних установок зависимостей: из коробки вы получаете все необходимое для общения, разработки и игр

Красивый интерфейс

Кастомная тема KDE Plasma с тёмными тонами и плавными анимациями для эстетического удовольствия.

Простота установки

Интуитивный установщик Calamares делает процесс установки доступным даже для новичков.

Безопасность

Встроенные механизмы безопасности и регулярные обновления для защиты вашей системы.

Высокая производительность

Оптимизированное ядро и минимальные накладные расходы обеспечивают максимальную отзывчивость.

Богатый набор ПО

Предустановлены все необходимые приложения для работы, мультимедиа и разработки.

Установка

1

Скачайте образ

Загрузите последнюю версию EN-OS с нашей страницы релизов.

wget https://en-os.org/downloads/en-os-latest.iso
2

Запишите на USB

Используйте BalenaEtcher или dd для записи образа на флешку.

dd if=en-os.iso of=/dev/sdX bs=4M status=progress
3

Установите систему

Загрузитесь с USB и следуйте инструкциям установщика Calamares.
